# ImageRail / 画轨

ImageRail 是一个轻量的图片版本管理桌面工具，用来把同一创作方向下的图片按轨道整理、命名、备注和对比。

## 下载使用

请在 GitHub Releases 页面下载已打包版本：

https://github.com/Star-Fish-C/Image-rail/releases

下载 `ImageRail.exe` 后直接运行即可。

普通用户不需要安装 Node.js、Rust 或 Tauri。

## 当前版本

当前版本是 Tauri 版 MVP，重点保留图片轨道管理的核心流程。

## 主要功能

- 创建和打开 ImageRail 项目
- 在项目里创建多条图片轨道
- 将本地图片或网页图片拖入轨道
- 自动复制图片到对应轨道文件夹
- 自动命名图片，例如 `A_1.png`、`A_2.png`
- 显示缩略图、文件名、版本号、备注和状态
- 支持状态标签：可用、待定、废弃、最终候选
- 点击图片后在右侧查看大图
- 支持固定一张图片，与另一张图片上下对比
- 支持鼠标滚轮缩放、左键拖动画布查看细节
- 支持重命名项目显示名、轨道文件夹和同轨道图片名前缀
- 删除图片时会删除应用记录和本地图片文件
- 删除轨道时会删除应用记录和本地轨道文件夹

## 项目文件结构

创建项目时，选择一个保存位置后，ImageRail 会先在该位置内创建一个项目文件夹，再在项目文件夹内创建轨道文件夹。

示例：

```text
你选择的文件夹/
└─ ImageRail_Project/
   ├─ track_A/
   │  ├─ A_1.png
   │  └─ A_2.png
   └─ track_B/
      └─ B_1.png
```

项目列表和项目数据保存在应用目录下的 `project-data` 文件夹中，不直接放进你选择的项目文件夹。
