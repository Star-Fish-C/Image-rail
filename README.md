# ImageRail / 画轨

ImageRail 是一个图片版本管理桌面应用，用来把同一方向的图片按轨道整理、命名、备注和对比。
平时会制作大量的同样内容不同版本的图片，用于管理和重命名

## 下载使用

请在 GitHub 仓库的 Releases 页面下载已打包版本：

https://github.com/Star-Fish-C/Image-rail/releases

下载 `.exe` 后直接运行即可，不需要安装 Node.js，也不需要手动配置环境。

## 主要功能

- 选择一个项目文件夹
- 在项目文件夹内创建图片总文件夹，默认名为 `images`
- 新建多条图片轨道
- 将图片拖入轨道
- 自动复制图片到项目文件夹内
- 自动命名图片，例如 `A_1.0.png`、`A_1.1.png`
- 显示图片缩略图、文件名、版本号
- 为每张图片添加备注
- 为每张图片设置状态：可用、待定、废弃、最终候选
- 点击图片放大预览
- 删除图片记录，但不直接删除硬盘原文件
- 删除轨道记录，但不直接删除硬盘轨道文件夹和图片
- 保存项目数据，下次重新选择同一项目文件夹可继续管理

## 文件结构

选择项目文件夹后，默认结构类似：

```text
项目文件夹/
└─ images/
   ├─ track_A/
   │  ├─ A_1.0.png
   │  └─ A_1.1.png
   └─ track_B/
      └─ B_1.0.png
```

项目数据不会直接放在你选择的项目文件夹里，而是保存在应用目录下的 `project-data` 文件夹中。

## 基本使用

1. 打开 ImageRail。
2. 点击“选择项目文件夹”。
3. 选择一个已有文件夹，或新建一个项目文件夹。
4. 点击“新建轨道”。
5. 将图片拖入轨道。
6. 在图片卡片中填写备注、选择状态。
7. 点击图片缩略图可放大预览。

## 重命名

- 顶部“重命名图片文件夹”：重命名项目文件夹里的图片总文件夹。例如项目是 `E:\SFC\测试`，默认图片文件夹是 `E:\SFC\测试\images`，可以改成 `E:\SFC\测试\分镜头`。不会修改外层的 `测试` 文件夹。
- 轨道左侧“重命名轨道”：重命名图片总文件夹里的轨道文件夹，并更新图片路径记录。
- 轨道左侧或图片卡片里的“重命名前缀”：修改同一条轨道里所有图片的真实文件名。例如把前缀 `A` 改成 `hero` 后，图片会变成 `hero_1.0.png`、`hero_1.1.png`、`hero_1.2.png`。

如果重命名失败，通常是文件夹或图片正在被占用。请关闭资源管理器、图片查看器、Photoshop、ComfyUI 等可能正在打开该文件夹的软件后再试。

## 删除说明

当前版本的删除功能偏安全：

- “移除记录”只删除应用里的图片记录，不删除硬盘图片文件。
- “删除轨道”只删除应用里的轨道记录，不删除硬盘轨道文件夹和图片。

真实删除硬盘文件的功能会在后续版本中单独设计。

## 当前版本

当前版本是 MVP，已包含基础图片轨道管理、拖拽导入、自动命名、备注、状态标签、放大预览、重命名和轨道删除。

双图对比等高级功能还在规划中。
